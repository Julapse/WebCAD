# WebCAD 静态试用版

访问地址：

https://julapse.github.io/WebCAD/

这是低成本静态前端试用版，用于快速体验当前 CAD 工作流。当前版本无账号、无云端保存、无服务器备份。

## 重要说明

- 请勿上传敏感或保密 CAD 文件。
- 当前文件只存在于浏览器页面内存中。
- 刷新或关闭页面前，请手动保存 `.cax` 文件到本地。
- 下次继续试用时，通过“打开”重新载入本地 `.cax` 文件。
- 建议使用最新版 Chrome 或 Edge 桌面浏览器。

## 已验证试用流程

1. 打开 F12 -> Console 并清空现有日志。
2. 刷新页面。
3. 新建草图 rectangle。
4. 拉伸，建议先用 10 或 20（首次冷启动调内核较慢，之后正常操作即可）。
5. 倒圆角。
6. 保存 `.cax`。
7. 刷新页面。
8. 打开刚保存的 `.cax`。
9. 导出 STEP。

## 当前边界

- 不提供账号系统。
- 不提供云端文件保存。
- 不提供多人协作或分享。
- 不提供后端 Geometry Service。
