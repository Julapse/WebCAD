# WebCAD 静态试用版

访问地址：

https://julapse.github.io/WebCAD/

这是低成本静态前端试用版，用于快速体验当前MVP版CAD工作流。当前版本无账号、无云端保存、无服务器备份。

## 重要说明

- 请勿上传敏感或保密 CAD 文件。
- 当前文件只存在于浏览器页面内存中。
- 刷新或关闭页面前，请手动保存 `.cax` 文件到本地。
- 下次继续试用时，通过“打开”重新载入本地 `.cax` 文件。
- 建议使用最新版 Chrome 或 Edge 桌面浏览器。

## 已验证试用流程

1. 进入网站，默认有一个文档（未命名.cax）。
2. 创建草图：绘制矩形或圆，完成草图。
3. 拉伸：选择草图拉伸为体。
4. 倒圆角：选择体的边。
5. 保存 `.cax`。
6. 刷新页面，重置浏览器。
7. 打开刚保存的 `.cax`。
8. 导出 STEP。

## 当前边界

- 不提供账号系统。
- 不提供云端文件保存。
- 不提供多人协作或分享。
- 不提供后端 Geometry Service。
- 不提供服务器端任务队列、日志、备份或权限管理。
